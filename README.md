# Smart Home REST API

Node.js/Express backend for a smart home system. Supports both MongoDB (optional) and an in-memory mock database defined in `data.js`.

## Features

- User registration and login (basic auth header helper)
- Houses, rooms, and devices (add/remove/update)
- Manual device control and status
- Room temperature and lighting controls
- Security device state updates
- Automations and scenarios
- Widgets and notifications
- Basic authentication middleware
- Centralized error handling and structured responses

## Tech Stack

- Node.js
- Express
- MongoDB + Mongoose (optional)
- Helmet, Morgan
